1. В чем разница между null и undefined?

Undefined («неопределенный») представляет собой значение по умолчанию:

  • переменной, которой не было присвоено значения, т.е. объявленной, но не инициализированной переменной;
  • функции, которая ничего не возвращает явно, например, console.log(a);
  • несуществующего свойства объекта.
let a
const doNothing = () => {} 
const someObj = { 
    a: 1,
    b: 'abc',
    c: false 
} 

console.log(a) // undefined 
console.log(doNothing()) // undefined 
console.log(someObj['d']) // undefined

Null — это «значение отсутствия значения». Null — это значение, которое присваивается переменной явно.

let b = 'null'

console.log(b) // null

2. Почему вывод в консоль obj.someprop.x приводит к ошибке?

Мы пытается получить доступ к свойству x свойства someprop, которое имеет значение undefined. obj.__proto__.__proto = null, поэтому возвращается undefined, а у undefined нет свойства x.

const obj = {}

console.log(obj.someprop.x) // Uncaught TypeError

3. Как проверить, является ли значение массивом?

Используем метод Array.isArray() возвращает true, если объект является массивом и false, если он массивом не является.

// Все следующие вызовы вернут true

Array.isArray([])
Array.isArray([1])
Array.isArray(new Array())

// Малоизвестный факт: Array.prototype сам является массивом:

Array.isArray(Array.prototype)

// Все следующие вызовы вернут false

Array.isArray()
Array.isArray({})
Array.isArray(null)
Array.isArray(undefined)
Array.isArray(17)
Array.isArray("Array")
Array.isArray(true)
Array.isArray(false)
Array.isArray({ __proto__:Array.prototype })

4. Проверить, является ли целое число квадратом, не используя математические функции.

Для этого напишем функцию в основе которой простой цикл for. В цикле проходимся по всем числам от 0 до n. Далее идет проверка: если какое-то из перечисленных чисел при умножении само на себя дает n, значит n является квадратом числа i и возвращаем true. Во всех остальных случаях возвращаем false. Также следует указать краевое условие для чисел 0 и 1, чтобы код работал корректно. В таком формате идет охват уже всех чисел от 0 до n. 

let isSquare = function(n) {

    if(n === 0 || n === 1) {
        return true
    }

    for (let i = 0; i < n; i++) {
        if(i * i === n) {
            return true
        }
    }

    return false
}

5. Написать функцию, которая принимает в качестве аргумента строку и возвращает значение true, если строка является палиндромом, и false, если это не так.

Для этого напишем следующую функцию. Предварительно приведем к общему регистру (метод toLowerCase или toUpperCase) и уберем все пробелы при помощи метода replace и регулярного выражения. Далее сравниваем значение str с этимже значением, только в отзеркаленном виде. Зеркалим при помощи методов: split (преобразуем строку в массив посимвольно), reverce (меняем порядок следования массива) и join (преобразуем полученный массив обратно в строку). Если строка является палиндромом, получаем true, иначе false. 

palindrome('racecar') // true
palindrome('table') // false
palindrome('Anna') // true

function palindrome(str) {
    str = str.toLowerCase().replace(/\s/g,'')
    return str === str.split('').reverse().join('')
}

Если строка длинная, то можно решить задачу через цикл for. Для этого делим длину строки пополам, чтобы не сравнивать повторно символы по которым уже прошлись. Далее проверяем равен ли первый символ последнему и т.д. до середины. Если несоответвий нет, значит строка является палиндромом true, иначе false 

palindrome2('А роза упала на лапу Азора') // true

function palindrome2(str) {
    str = str.toLowerCase().replace(/\s/g,'')
    const len = Math.floor(str.length / 2)

    for(let i = 0; i < len; i++) {
        if(str[i] !== str[str.length - i - 1]) {
            return false
        }
        return true
    }
}

6. Напишите функцию, которая возвращает n-ую запись в последовательности Фибоначчи, где n — это число, которое вы передаёте в качестве аргумента функции.

Здесь также есть два решения. В первом мы сохраняем все числа в массив result. Далее в цикле for итерируемся с 2. Определяем предыдущее и предпредыдущее значение переменной, а их сумму пушим (push) в массив. Когда мы дошли по значения заддыным аргументов num - выводим результат. 

function fibonacci(num) {

    const result = [ 0, 1 ]

    for (let i = 2; i <= num; i++) {
        const prevNum1 = result[i - 1]
        const prevNum2 = result[i - 2]

        result.push(prevNum1 + prevNum2)
    }

    return result[num]
}

Во втором работаем только с последними двумя числами. Создадим две переменные a и b. В этих переменных будем хранить предыдущее и предпредыдущее значение Фибоначчи. В цикле for итератор ставим на 3 позицию, т.к. первые 2 числа изветны. Далее переопределяем значение переменных ( a присваеваем значение b, а b присваеваем сумму значений a + b ) 

function fibonacci2(num) {

    let a = 1
        b = 1

    for (let i = 3; i <= num; i++) {
        [ a, b ] = [ b, a + b ]
    }

    return b
}
